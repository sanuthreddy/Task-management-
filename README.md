# Web Application

A Node.js and Express.js-based web application with EJS templates for dynamic pages. Includes user authentication, secret sharing, and basic styling.

## Features
- User Registration and Login
- Secret Sharing
- Responsive Design with CSS

## File Structure
- **`index.js`**: Main server file.
- **EJS Templates**: `header.ejs`, `footer.ejs`, `home.ejs`, `login.ejs`, `register.ejs`, `secrets.ejs`, `submit.ejs`, `submitsecret.ejs`, `new.ejs`.
- **CSS Files**: `main.css`, `styles.css`.
- **Other Files**: `package.json`, `package-lock.json`, `solution.js`, `newfiledup.js`.

## Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   node index.js
   ```
4. Open `http://localhost:3000` in your browser.
